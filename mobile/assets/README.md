# Assets

This directory contains the app's image assets.

## Required Files (Replace with actual assets)

- `icon.png` - App icon (1024x1024)
- `splash-icon.png` - Splash screen icon (200x200)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)
- `notification-icon.png` - Push notification icon (96x96)

## Design Guidelines

- Use a deep cosmic purple (#0D0B2E) background
- Gold (#F59E0B) accents for the logo
- Consider a stylized planetary/celestial design
- Include Sri Lankan cultural motifs subtly
